Instrucciones de uso para las pruebas del código

1. Buscar archivo llamado "Pruebas_195145245_VinalsLenero.txt"
2. Escoger una de las pruebas indicadas
	(Están indicadas como "Prueba Defecto", "Prueba 1", etc)
3. Copiar las lineas de codigo correspondiente entre cada titulo, estan separadas por guiones "-"
4. Pegar las lineas en la parte inferior del código, está comentado al final como [ESPACIO PARA PRUEBAS]
	(El archivo del código se llama "laboratorio_195145245_VinalsLenero.rkt")
5. Correr el codigo

Observacion
las pruebas son funcionas que se llaman, cada prueba tiene las mismas funciones con distintos parametros
el poner más de una prueba en el código generará problemas por conflicto de nombres, para tener en consideración
